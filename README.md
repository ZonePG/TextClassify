# TextClassify
HFUT Soft Engineering Competition.

## Requirements
```
pip install -r requirements.txt
```

## How to use
`cd` Current Directory
### On Windows
```
set FLASK_APP=run.py
set FLASK_DENUG=True
flask run
```
### On Mac/Linux 
```
export FLASK_APP=run.py
export FLASK_DENUG=True
flask run
```
